# Serv00 - 控制面板自动登录脚本
## 使用方法
　　在 GitHub 仓库中，创建一个名为 ACCOUNTS_JSON 的 Secret，将 JSON 格式的账号密码字符串作为它的值，如：  
[  
  { "username": "qinshihuang", "password": "jinwandalaohu" },  
  { "username": "zhaogao", "password": "daqinzhonggong" },  
  { "username": "heiheihei", "password": "shaibopengke" }  
]
